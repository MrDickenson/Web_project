Roadmap: ToDo System Lite

План розвитку функціонала на найближчий місяць.

Тиждень 1:

Реєстрація та Вхід: Створення форми логіну та реєстрації.

Приватність: Кожен користувач бачить виключно власні завдання.

Тиждень 2:

Редагування: Можливість виправити текст існуючого завдання.

Видалення: Кнопка для очищення списку від непотрібних записів.

Статус: Чекбокс для швидкого завершення справи.

Тиждень 3:

Фільтри: Перемикачі для перегляду тільки активних або завершених справ.

Сортування: Відображення списку за датою створення (нові зверху).
