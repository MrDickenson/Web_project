Глосарій:
1. User: Основна інформація (ім'я, email).

2. Profile: Налаштування користувача (тема, мова, часовий пояс).

3. Task (Завдання): одиниця роботи або дія, яку користувач планує виконати; основний елемент системи.

4. Project (Проєкт): логічний контейнер, що об'єднує групу завдань за спільною метою або сферою (наприклад, "Робота", "Дім").

5. Status (Статус): поточний стан життєвого циклу завдання (наприклад, "Нове", "В процесі", "Виконано").

6. Priority (Пріоритет): показник важливості або терміновості завдання, що впливає на порядок його виконання.

7. Tag (Тег): текстова мітка для наскрізної категоризації завдань із різних проєктів.
